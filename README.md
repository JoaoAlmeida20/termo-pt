# termo-pt

**Termo PT** is a European Portuguese version of [Wordle](https://www.nytimes.com/games/wordle/index.html), the popular word guessing game originally created by [Josh Wardle](https://www.powerlanguage.co.uk/).

Inspired by [Termo](https://term.ooo), a version of Wordle for Brazilian Portuguese created by [Fernando Serboncini](https://fserb.com).

## Word List

The word list is based on the [SUBTLEX-PT](https://www.psi.uminho.pt/pt/CIPsi/Laboratorios_Investigacao/Psicolinguistica/Paginas/Bases_Lexiais_Estudos_Normativos.aspx) lexicon for European Portuguese, derived from film and television subtitles, built by researchers at the Universidade do Minho:

> Soares, A. P., Machado, J., Costa, A., Iriarte, A., Simões, A., de Almeida, J. J., Comesaña, M., & Perea, M. (2015). On the advantages of frequency measures extracted from subtitles: The case of Portuguese. *Quarterly Journal of Experimental Psychology*, *68*, 680–696.
